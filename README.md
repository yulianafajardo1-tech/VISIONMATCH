👓 Gafas inteligentes con IA
Aplicación web desarrollada con Streamlit que permite a ópticas subir la foto de un cliente y, mediante inteligencia artificial, analizar su tipo de rostro para recomendar las gafas más adecuadas.
Además, la aplicación muestra una simulación en tiempo real de cómo se vería el cliente con diferentes monturas.
🚀 Funcionalidades
📸 Carga de imagen del cliente
🤖 Análisis facial automático
🧠 Clasificación del tipo de rostro (ovalado, redondo, cuadrado, etc.)
👓 Recomendación de gafas personalizadas
🪞 Visualización en tiempo real de monturas sobre el rostro
🛠️ Tecnologías
Pitón
Streamlit
OpenCV (procesamiento de imagen)
Modelos de Machine Learning / IA
📦 Uso
Subir una imagen del cliente
La aplicación analiza el rostro automáticamente.
Se muestran recomendaciones de gafas.
Se visualiza la montura en el rostro
🎯 Objetivo
Ayudar a las ópticas a mejorar la experiencia del cliente, agilizar la asesoría y aumentar las ventas mediante tecnología de inteligencia artificial.
🌐 Futuro
Integración con catálogos reales de gafas.
Sistema de usuarios paras
Implementación en tiendas físicas
